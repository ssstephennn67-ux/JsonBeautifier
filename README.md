# Smart JSON Studio

A modern, developer-centric JSON beautifier designed to handle complex data structures with ease.

## Key Feature: Smart Array Filtering

Stop scrolling through endless fields. **Smart JSON Studio** allows you to extract and toggle specific keys within any array of objects, turning cluttered data into a scannable list.

## How it works
1. **Parse**: Paste your JSON and click **Parse & Beautify**.
2. **Filter**: Locate an array and click the **Filter** button.
3. **Select**: Check the fields you need and hit **Apply Filter**.
4. **Focus**: Analyze your data without the noise.
